Interpretation note
-------------------
These runs are language-specific held-out evaluations with train-split-only normalization and train-split-only ranking.
Task-specific features are sparse by design, so task-only results can be dominated by which task domains exist within each language.
For k=all, ANOVA and Welch use the same usable feature pool; any difference would indicate a resume/version issue rather than ranking behavior.
