# Submission Guidelines

## File Naming Convention
You may send up to five sets of results to us for scoring for each task. Please ensure that you follow the file naming convention:
- Task 1: "taukadial_results_task1_attempt1.txt", "taukadial_results_task1_attempt2.txt", ..., "taukadial_results_task1_attempt5.txt"
- Task 2: "taukadial_results_task2_attempt1.txt", "taukadial_results_task2_attempt2.txt", ..., "taukadial_results_task2_attempt5.txt"

## Contents of TXT Files
Ensure that the submitted TXT files contain the following information:
- Task 1: Predicted label (DX either MCI/NC)
- Task 2: MMSE score

## Email Subject Format
When submitting your results, please make sure to mention the evaluation of your method and team/team lead name in the email subject line.

Example:
Subject: [Your Method Name] - [Team Lead/Team Name]
