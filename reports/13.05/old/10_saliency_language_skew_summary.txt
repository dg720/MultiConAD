Saliency Radar Interpretation
============================
Method
------
These radar maps use AD-oriented feature burden scores on a 0-100 scale.
For each feature, the direction is set from the training split so that higher values are more AD-like.
The fixed-feature plots use the same top 16 pooled all-universal features in every language.
The local-feature plots use the top 16 features from the best all-universal model within each language.

Interpretation
--------------
The fixed pooled feature set does show language-specific skew rather than one universal saliency shape.
English and Spanish lean more on lexical, syntactic, and discourse burden.
Chinese shows stronger pause/rate and some acoustic burden on the pooled feature set.
Greek remains the most acoustically skewed profile even when the feature set is fixed across languages.
