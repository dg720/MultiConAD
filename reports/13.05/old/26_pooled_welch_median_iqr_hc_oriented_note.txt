Pooled Welch Median + IQR Overlay (HC-Oriented Scale)
====================================================
This plot uses the same fixed pooled universal Welch feature set as the standard median/IQR version.
The only change is score direction:
- original burden scale: higher = more AD-like
- reversed burden scale: higher = more HC-like
- mathematically: `reversed_score = 100 - burden_0_100`
